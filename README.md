# Features included in this project:
* user features:

* register and login users.

* post images can be uploaded using the file system.

* pagination on every page.

* save any post to the collection.

* saved posts page.

* delete posts.

* Password is stored in the database in encrypted format with salt.

* create and edit posts.


# Admin features:


* admin can create their own accounts and posts.

* admin can delete their own posts.

# What you need to run this application:

* Node
* MongoDB or Mongo Atlas
* NPM
* How to run this application

# Make sure MongoDB is running on your system or online.
include the MongoDB database link in the .env file.
Add cloudinary data in /client/src/utils/imageUpload.js file.
Clone this repository
Open the command line in the cloned folder,
* To install dependencies for the backend, run npm install in the main folder.
* To run the backend type the command node server in the main folder.
* To install dependencies for front end, run npm install in /client folder.
* To run frontend type command npm start in /client folder.
* Open localhost:3000 in the browser. And application will be up and running. Here are some images of the application:
